# Freeplane

## Create a template for mind maps

- `File` > `New map from template...` > Select the base template you want to use
- `Format` > `Manage Styles` > `Edit Styles` 
- Now you can customize the styles

## Add node details

Right-click on a node > `Edit` > `Edit node details in dialog` OR `Edit node details in-line`

## Add a note to a node

Right-click on a node > `Edit` > `Edit note in dialog`

## Turn off automatic node collapsing

`Freeplane` > `Settings...` > `Behaviour` > Uncheck `Fold on click inside`

## Format Text Within a Node

 This allows you to apply unique styles—such as bolding, italics, custom text colors, 
 or font sizes—to specific characters, words, or paragraphs without changing the rest of the node.
 
Select the node ➔ `Edit` ➔ `Edit node core in dialog`

OR for in line editing

Select the text part to be formatted ➔ Right Click ➔ `Format` ➔ ** required formatting option 

## Set Markdown as the Default Formatting Style

To update your current map's core template:

   1. @main top menu, `Format` -> `Manage Styles` -> `Edit styles` 
   2. Select the Base Style: A template sub-map will open. Look for the node named Default (this acts as the master template for all nodes).  
   3. With the Default node selected, open the Properties Panel on the right side of your screen. 
Look for the Core Text / Format section and change its dropdown setting to `Markdown`.  
   4. click `OK` to save and exit
