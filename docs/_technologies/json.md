# JSON
`data interchange format`

```json
{
      "name": "John Doe",
      "age": 30,
      "isStudent": false,
      "address": {
            "street": "123 Main St",
            "city": "Springfield",
            "state": "IL",
            "postalCode": "62701"
      },
      "hobbies": ["Reading", "Cycling", "Photography"]
}
```

* Comments are not supported
* `address` is a **dictionary**
* `hobbies` is a **list**
