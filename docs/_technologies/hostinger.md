# Hostinger

## Shared web hosting

- Many sites on the same server
- Hostinger handles 
  - server maintenance
  - control via hPanel
  - one-click installers (WordPress) 
  - managed environment

## VPS hosting

- You get a virtualized server slice with dedicated RAM/CPU/storage
- You can run custom software, choose OS, and (depending on plan) have root/SSH access
- Typically more responsibility for maintenance unless you choose managed VPS options
