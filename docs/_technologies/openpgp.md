# OpenPGP

`a standard for public-key encryption and signing of data and email`

Implementations  
- GnuPG (`gpg`)
- OpenPGP.js
- Sequoia
- rnp
- BouncyCastle

Use cases
- encrypt files/email
- digitally sign messages
- verify integrity/authenticity
- key exchange
