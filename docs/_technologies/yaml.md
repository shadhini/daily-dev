# YML

`data serialization format`

```yml
name: "John Doe"
age: 30
is_student: false
address:
      street: "123 Main St"
      city: "Springfield"
      state: "IL"
      postal_code: "62701"
hobbies:
      - "Reading"
      - "Cycling"
      - "Photography"
```

* Comments start with #
* `address` is a **dictionary**
* `hobbies` is a **list**
